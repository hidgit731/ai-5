# Сервис хранения прогресса студента

---

## Локальная разработка

### Установка зависимостей

1. Обязательно нужно собрать образы
```bash
make build
```

2. Установка зависимостей
```shell
make composer_install
```

### Взаимодействие с сервисом
- Запуск сервиса. После проверьте работу, перейдя на http://localhost:7777
```shell
make up
```

- Остановка сервиса
```shell
make down
```
